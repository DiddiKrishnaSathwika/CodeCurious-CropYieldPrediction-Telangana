# Dataset Details

We used a crop production dataset and filtered it for Telangana.

## Raw Dataset
Stored in:
`data/raw/india_crop_production.csv`

## Processed Dataset
Stored in:
`data/processed/telangana_crop_data.csv`

## Main Fields Used
- state
- district
- crop
- area
- production

## Added Prototype Fields
- rainfall
- temperature
- humidity
- soil_type
- soil_ph
- water_availability

These were added to make the prototype compatible with the prediction workflow.
