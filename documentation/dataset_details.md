# Dataset Details

## Raw Dataset

We used a crop production dataset and filtered it for Telangana.

Location:
data/raw/india_crop_production.csv

## Processed Dataset

After filtering and preprocessing, the dataset is saved at:

data/processed/telangana_crop_data.csv

## Features Used

- crop_type
- district
- rainfall
- temperature
- humidity
- soil_type
- soil_ph
- water_availability
- area

## Target Variable

- yield

## Note

Some environmental features like rainfall, temperature, and soil details are added for prototype purposes and can be replaced with real data in future.
